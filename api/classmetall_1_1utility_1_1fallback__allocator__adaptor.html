Detailed Description

template<typename primary_alloc>
class metall::utility::fallback_allocator_adaptor< primary_alloc >

A STL compatible allocator which fallbacks to a heap allocator (e.g., malloc) if its constructor receives no argument.

Template Parameters
primary_allocA primary allocator type, i.e., metall::stl_allocator.

Using this allocator, the following code will work:

{
using alloc = fallback_allocator_adaptor<metall::manager::allocator_type<int>>;
vector<int, alloc> temp_vec; // Allocate an vector object temporarily in 'DRAM'.
}
Attention
The purpose of this allocator is providing a way to quickly integrate Metall into an application which wants to allocate 'metallized' classes temporarily in 'DRAM' occasionally. This allocator could cause difficult bugs to debug. Use this allocator with great care.
Examples
fallback_allocator_adaptor.cpp.
